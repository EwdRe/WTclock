# WTclock 电子天气时钟
---
给老windows平板写了个程序，显示时间天气，待办事项，宿舍电费等

基于Python3.9界面用的PyQT5[![Python](https://img.shields.io/badge/-Python-3776AB?style=flat&logo=python&logoColor=white)](#)

目前适用于1080P  打开程序后点击左下角的菜单按钮即可启动时钟 退出按钮是左下角第二个
